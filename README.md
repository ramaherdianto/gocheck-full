
# Todo App

Todo App with Sorted tasks feature using React JS and TailwindCSS




## Run Locally

Clone the project

```bash
  git clone https://github.com/ramaherdianto/gocheck-full.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
